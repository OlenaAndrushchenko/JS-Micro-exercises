Exercises

  1. Write a program that writes a text of type <h1> on the screen that says “Hello Javascript”.
  2. Write a program that writes on the screen “The sum of 3 + 5 is <result>”.
  3. Write a program that asks for the user's name with an input and writes a text that says “Hello <user-name>”.
  4. Write a program that asks for two numbers and writes “The sum of <number-one> with <number-two> is <result>”.
  5. Write a program that asks for two numbers and writes on the screen which one is the largest.
  6. Write a program that asks for 3 numbers and writes the largest of the three on the screen.
  7. Write a program that asks for a phrase and writes how many times the letter 'a' appears.
  8. Write a program that asks for a sentence and writes the vowels that appear.
  9. Write a program that asks for a sentence and writes how many vowels there are in the sentence.
  10. Write a program that asks for a sentence and writes how many times each of the vowels appears.
  11. Write a program that asks for a number and tells us if it is divisible by 2, 3, 5, or 7 (check if it is divisible by one of the four).
  12. Add to the previous exercise that tells us by which of the four it is divisible (state all by which it is divisible).
  13. Write a program that writes on the screen the divisors of a given number.
  14. Write a program that writes on the screen the common divisors of two given numbers.
  15. Write a program that tells us if a given number is prime (it is not divisible by any number other than itself or the unit).
